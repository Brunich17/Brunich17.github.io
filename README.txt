Proyecto Botica Marmedic E&G

Para usar:
1. Abre index.html con Live Server en Visual Studio Code.
2. Coloca tus imágenes reales en la carpeta img/ y img/productos/.
3. Si agregas productos, edita data/productos.js.
4. Para subir a GitHub Pages, sube toda esta carpeta al repositorio.

Nota: Si una imagen no existe, se mostrará una imagen temporal.
